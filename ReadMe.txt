Potrebno instalirati:
npm init
npm install express
npm chai
npm chai-http
npm mocha
npm mysql
npm mysql2
npm sequelize

Testovi sa spirale 3 se pokrecu kucanjem u terminal:
mocha

Posljenji zadatak spirale 3 je podešen da radi sa bazom.
Da bi ponovno radio sa datotekama potrebno je otkomntarisati u html-u komentare, te zakomentarisati script unosRasporedaS4.js

Najviše korišteni fajlovi za spiralu 4:
server.js,db.js,unosRasporedaS4.js,studenti.js,studenti.css,studenti.html

Postavke zadataka su u folderu RequestedTasks.
